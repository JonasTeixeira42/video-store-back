# Consulta de todos os Diretores

Retorna todos os diretores cadastrados na paltaforma.

- **PATH** <br />
  /directors

- **Method** <br />
  `GET`

- **Success Response**

  - **Code**: 200 <br />
    **Description**:
    ```json
    [
      {
        "id": 1,
        "name": "xxxx"
      }
    ]
    ```
